# ⚡ Smart Energy Saver Website

A web-based application designed to help users monitor and reduce their home electricity usage by providing energy-saving tips, tracking usage patterns, and improving efficiency.

---

## 🚀 Features
- User authentication (Login/Register)
- Dashboard for energy insights
- Energy-saving tips and suggestions
- Usage tracking system
- Admin panel for management
- Responsive UI (Bootstrap)

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, Bootstrap
- Backend: PHP (OOP / MVC structure)
- Database: MySQL
- Server: Apache (Laragon/XAMPP)
